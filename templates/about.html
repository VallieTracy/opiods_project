Opioid Epidemic Visualization Project

This website was created to present our work completed for Project Two in the Data Analytics and Visualization Coding boot camp offered through The University of Minnesota, College of Continuing Education and Professional Studies. For our topic we choose to analyze and visualize data from 200-2018 about the Opioid Epidemic in the United States. Our interactive dashboard displays data collected by the Center for Control Disease Control detailing the death rate per 100,000 by opioid overdose and prescription drug sales of Oxycodone and Hydrocodone for 100,000. Currently this webpage draws the data from a SQLite database and is hosted on Heroku.com. The visualizations on this page present several interactive graphics to help the user navigate through the data set. The graphics leverage Leaflet, MapBox, and Amcharts Javascript libraries. Bootstrap libraries were used to format the website. We have also made out data available in an API route for so that students and researchers can easily access this data.
